# Colors in CSS

## What is Color?

### The color property allows you to specify the color of text inside an element.


## Colors in CSS is one of three ways:

- Color name ( color: black;)

- HEX codes ( color:  #ee3e88;)

- RGB values ( color:  rgb(100,50,90); )
